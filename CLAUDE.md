# QAMFSHOP — QubicaAMF Official Merch Store

## Project Overview

Single-page e-commerce storefront for QubicaAMF bowling merchandise. Static HTML/CSS/JS deployed to GitHub Pages with Printify print-on-demand fulfillment.

## Architecture

- **Single file SPA**: `index.html` (3,400+ lines) contains all CSS, HTML, and JS
- **Three views**: Store (`#store`), Marketing (`#marketing`), Admin (`#admin`)
- **Hash-based routing**: No backend — fully client-side
- **Data**: `products.json` (Printify sync) + `custom-products.json` (manual products)
- **Persistence**: localStorage for cart, wishlist, admin settings, custom products

## Key Files

| File | Purpose |
|------|---------|
| `index.html` | Entire storefront, marketing page, and admin panel |
| `products.json` | Auto-generated product catalog from Printify API |
| `custom-products.json` | Manually maintained custom fulfillment products |
| `sync-printify.js` | Node.js script to fetch products from Printify API |
| `.github/workflows/deploy-pages.yml` | Auto-deploy to GitHub Pages on push to main |
| `.github/workflows/sync-printify.yml` | Daily Printify product sync at midnight UTC |

## Development

```bash
npm install              # Install dotenv dependency
npm run sync:demo        # Generate demo product data
npm run sync             # Sync from live Printify API (requires .env)
```

## External Services

- **Printify**: Print-on-demand fulfillment (API token + shop ID in .env)
- **Stripe**: Payment processing (planned, not yet integrated)
- **GitHub Pages**: Static hosting (auto-deploys from main)
- **BesXtras.com**: WordPress blog feed (fetched client-side via WP REST API)

## Important Rules

- **Never edit `.env`** — contains API secrets
- **Never edit `products.json` manually** — it's auto-generated by sync script
- **Never edit `package-lock.json`** directly
- Pushing to `main` triggers a production deploy — always verify changes first
- Admin password hash is in JS — default is "password" (SHA-256 hashed)
- The `index.html` file uses CSS nesting (`#page-store { ... }`) — requires modern browsers

## Design System

- **Fonts**: Syne (display/headlines), Outfit (body/UI)
- **Primary color**: `--red: #E8192C`
- **Dark base**: `--dark: #0A0A0B`
- **Aesthetic**: "Midnight Lanes" — dark luxury athletic with bowling alley neon accents
